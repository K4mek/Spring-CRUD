# Spring-CRUD1
A **C.R.U.D** web application made with **Spring Framework** and **Postgresql** or **Mysql** as database and deployed with **Heroku**

Available in **https://userregistrationcrud.herokuapp.com/**

Technologies used were:
* Html, Css and Js
* Bootstrap 4
  * Grid system
* Spring Framework
  * Spring Boot
  * Spring Data JDBC
  * Thymeleaf engine
